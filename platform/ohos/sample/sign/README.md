# Local debug signing

Place your AGC debug files here (gitignored):

- `maplibre-debug.p12` — private keystore (generated locally with `hap-sign-tool`)
- `maplibre-debug.cer` — debug certificate from AppGallery Connect
- `maplibre-debug.p7b` — debug profile for bundle `org.maplibre.native.demo`

Keep signing passwords outside the repository.

## Configure local Hvigor signing

Copy `signing.local.example.json` to `signing.local.json`, keep the file
uncommitted, and fill in the encrypted password values generated by DevEco
Studio. The example uses paths under this directory so each developer can drop
their local debug files in the same place.

If DevEco Studio writes a signing config into `../build-profile.json5`, copy the
generated signing config object into `signing.local.json` and revert
`../build-profile.json5` before sending a PR.

## Build from the command line

Once `signing.local.json` is configured, the normal Hvigor build signs the HAP:

```sh
cd platform/ohos/sample
/path/to/command-line-tools/bin/hvigorw assembleApp --no-daemon
```
